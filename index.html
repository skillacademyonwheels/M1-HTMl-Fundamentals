HTML Structure

This is a simple HTML document to demonstrate the structure of an HTML file. It includes a title, a heading, and some text content.

HTML stands for HyperText Markup Language, and it is the standard markup language for creating web pages.


HTML elements are represented by tags, which are enclosed in angle brackets. For example, the <h1> tag is used for main headings, and the <p> tag is used for paragraphs.
